## 3. ALCANCE DEL PROYECTO

### Objetivo
Crear un sistema de monitoreo ejecutivo que observe las actividades de ODIN y genere automáticamente el Morning Brief diario para el CEO.

### Entregables MVP (Fase A)
1. Dashboard web de monitoreo ODIN
2. Sistema de Morning Brief automático (08:00 AM)
3. Login de seguridad para fundadores
4. Colectores de datos (Gmail, Calendar, GitHub, métricas)
5. Base de datos persistente (SQLite → PostgreSQL)
6. Sincronización con GitHub
7. Documentación técnica

### Fuera de Alcance (Fase B)
- Analytics avanzada de negocio
- UI mobile nativa
- Inteligencia predictiva
- Integración con fuentes externas de mercado
